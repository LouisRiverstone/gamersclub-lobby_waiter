# Uso

    Primeiramente você deve abrir seu Counter Strike pelo Anti-Cheat da GamersClub
    Depois disso aperte F12 no seu navegador na pagina de lobbys
    Abra a Aba 'Console'
    Cole o codigo do script.js
    aperte enter e seja feliz!
